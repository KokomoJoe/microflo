MicroFlo

Flow-based programming runtime for microcontrollers (Arduino)

MicroFlo is a Flow-based programming runtime for microcontrollers like the Arduino. It is inspired by and designed for integration with NoFlo, a JavaScript-based FBP system.

You can program with MicroFlo either:

  • Visually, using the NoFlo UI
  • Textually, using the declarative .fbp DSL or .json definition specced by NoFlo
  • Programatically, by embedding it and building a graph using the C++ API
